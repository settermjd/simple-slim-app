# Simple Slim App

Despite the name (well, this is a small [Slim app][slim-framework]) the purpose of this small PHP application is to demonstrate how to send emails using [Twilio SendGrid][twilio-sendgrid] in a [GitHub Actions][github-actions] workflow.
